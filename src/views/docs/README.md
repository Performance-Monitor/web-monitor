## 前端性能监控平台
该项目主要有以下功能:
* 前端首屏性能监控、性能分析
* `JS`性能监控
* 自动错误报警
* `pv/pu`收集
* `IP`转换工具
* 多个监测站点

### 前置条件
* 配置`node.js`环境
* 安装`MongoDB`数据库

### 依赖安装
```
* npm install
* cnpm install
* yarn install
```

### 运行项目
```
* npm run serve
* cnpm run serve
* yarn run serve
```

### 生产环境打包
```
* npm run build
* cnpm run build
* yarn run build
```

### 检错
```
* npm run lint
* cnpm run lint
* yarn run lint
```
